# Hardware

## Phone Mount — Smart V10

**File:** `PhoneMount_SmartV10.stl`

3D-printable mount designed to hold the Smart V10 controller phone on the hexapod robot frame. Allows the Hexapod Client app to be visible and accessible while the robot is operating.

**Print settings (recommended):**
- Material: PLA or PETG
- Layer height: 0.2 mm
- Infill: 20%
- Supports: as needed depending on orientation

**Compatibility:** dimensioned for the Smart V10 phone.
